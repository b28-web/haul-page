## Scan to Schedule Page

Build the `/scan` page — the destination for QR codes on printed materials (truck wraps, flyers, door hangers, the poster tear-off tabs). Two jobs: convert the scanner into a lead, and build trust with social proof.

## Scope

- Streamlined schedule CTA (call button + link to `/book`)
- Before/after photo gallery (operator uploads, not user-generated)
- Customer endorsements / testimonials
- Same dark theme, same typography
- LiveView (gallery content is dynamic, loaded from config or database)
- QR code generation for the operator's `/scan` URL (for print materials)
